Topic 2: Basic Korean sentence structure

In this lesson, you will learn to make a basic Korean sentence and understand the fundamental Subject-Object-Verb (SOV) order that differs from English.

Korean follows a Subject-Object-Verb (SOV) structure, which is different from English and Chinese that use Subject-Verb-Object (SVO) order. This means the verb always comes at the end of the sentence in Korean.

Korean (SOV)

저는 사과를 먹어요
Subject Object Verb

English (SVO)

I eat an apple
Subject Verb Object

Breaking Down the Structure

저는 사과를 먹어요. I eat an apple.

Subject (주어)

저는

저 = I/me (humble form)

는 = Topic marker

Indicates what the sentence is about

Object (목적어)

사과를

사과 = Apple

를 = Object marker

Marks the receiver of the action

Verb (동사)

먹어요

먹다 = To eat (dictionary form)

먹어요 = Eat (polite present tense)

Always comes at the end of sentence

More Examples

아진은 학교에 가요.
아진은 학교에 가요
Ajin goes to school.
아현은 영화를 봐요.
아현은 영화를 봐요
Ahyeon watches a movie.
저는 한국어를 공부해요.
저는 한국어를 공부해요
I study Korean.

Key Points to Remember

  • ①
    Verb always comes last - This is the most important rule in Korean sentence structure
  • ②
    Particles mark function - 은/는 (topic), 을/를 (object), 에 (location) help identify sentence components
  • ③
    Flexible word order - While SOV is standard, Korean allows some flexibility as long as the verb stays at the end
  • ④
    Subject can be omitted - When context is clear, the subject is often left out in conversation

Building on This Foundation

While SOV forms the core structure, Korean sentences can include additional elements like adjectives, adverbs, time expressions, and multiple objects. These typically follow the pattern: Time + Subject + Object + Verb. You'll learn about adding these components in future chapters.
